Mi Primer Commit en Git

  • Crear una carpeta vacia
  • En VS Code, abrir la carpeta creada (open folder)
  • Usar el menú de Terminal y seleccionar la opción New Terminal
  • En la terminal, escribir el comando git init
  • Se crea una subcarpeta 01-git y dentro un archivo index.html
  • Verifico el estado actual de git, con el comando git status
